# Электрик

Текущий проект реализован как `React 18 + TypeScript + Vite` SPA с локалями `ru` и `hy`.

## Команды

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Переменные окружения

Для корректных canonical URL и Open Graph укажите домен сайта:

```bash
VITE_SITE_URL=https://example.com
```

См. файл [.env.example](/Users/david/мои проекты/web/Electro/.env.example).

## Структура

- `src/` — актуальный фронтенд-код приложения.
- `public/` — статические файлы.
- `docs/` — продуктовые и архитектурные заметки.
- `blueprint/` — ранние контентные и архитектурные заготовки, не полностью совпадающие с текущей реализацией.

## Важно

Документы в `docs/implementation-blueprint.md` и `blueprint/` описывают целевое направление развития и содержат идеи под следующий этап. Они не являются точным отражением текущего состояния Vite-проекта.
